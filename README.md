# Online-Exam-System
Online Exam System using Python Flask
Online Exam System

This is a web based Online Examination System developed using Python Flask and SQLite.

Features

- Student Login & Registration
- Admin can add questions
- Exam timer
- Automatic result generation

Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite

Run Project

pip install flask
python app.py

Open browser:
http://127.0.0.1:5000
